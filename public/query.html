What will it teach me?

To work with objects and arrays, the main data structures in JavaScript. Most of the HTML, CSS and JavaScript you write must account for data extracted from objects. The more familiar you become with writing expressions that access values from objects, the better you'll feel when working with APIs as part of a real project.
